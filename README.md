# Lanchonete Bot

## Descrição
Este é um bot de WhatsApp para gerenciar pedidos de uma lanchonete. Ele inclui funcionalidades como:
- Cardápio dinâmico
- Registro de pedidos
- Painel administrativo
- Notificações automáticas
